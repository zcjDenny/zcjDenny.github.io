GET 验证设备状态

https://deepcast.top/api.html?action=verify&device_id=设备ID

参数说明

参数 类型 必填 说明
action string 是 操作类型，固定为"verify"
device_id string 是 设备唯一标识符

响应示例

{"status": "success",
"registered": true,
"start_date": "2025/9/22",
"days_remaining": 5,
"valid": true}

GET 注册新设备

https://deepcast.top/api.html?action=register&device_id=设备ID

参数说明

参数 类型 必填 说明
action string 是 操作类型，固定为"register"
device_id string 是 设备唯一标识符

响应示例

{"status": "success",
"message": "设备注册成功",
"start_date": "2025/9/22",
"days_remaining": 6}

使用说明

  1. 软件启动时会自动调用验证接口检查设备状态
  2. 如果设备未注册，会自动调用注册接口
  3. 试用期为6天，过期后需要输入授权码
  4. 所有验证数据存储在浏览器的localStorage中
  5. 支持离线验证（基于最后一次成功验证的结果）
